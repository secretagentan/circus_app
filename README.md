# Working in Rails

# Setup

Download dependencies

```
$ bundle install
```

Setup database

```
$ rails db:create
$ rails db:migrate
$ rails db:seed
```

# Adding dependencies

```
# Gemfile
gem 'annotate'
gem 'pry-rails'
# remember to run bundle install again!
```

# Run commands with bundle exec!

```
$ bundle exec annotate
```
